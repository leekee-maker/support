隐私说明

API Key 由系统钥匙串加密保护，仅在调用你配置的模型服务商时使用。

为什么需要配置 API Key？ 嗨聊通过你配置的模型服务商生成回复，API Key 用来完成请求认证。
图片为什么可以发送？ 部分模型支持视觉输入；不支持视觉的模型会先走本机文字识别，再生成回复。
聊天记录会上传吗？ 聊天数据默认保存在本机，不会自动上传到第三方服务器。

如需替换正式支持邮箱、帮助文档链接或隐私政策链接，直接改这里即可。
